# 2dActorToolsユーザーガイド
## 2dActorTools
合成音声ソフトの作成支援を行うPremiere Proのエクステンションです。  
[https://github.com/wabimochi/2dActorTools](https://github.com/wabimochi/2dActorTools){.new_tab}

## 主な機能
- 立ち絵の差分変更
- 音声データの自動インポート
- 字幕の配置
- 自動改行

## 動作環境
- WindowsとMacOSで使用できます。
- 基本的に最新のPremiere Proで使用することを想定しています。
- 現在は日本語環境のみ対応しています。
